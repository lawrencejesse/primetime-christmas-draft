# Prime Time Christmas Colt Draft 2025

## Overview
A static HTML website for the Prime Time Rodeos Christmas Colt Draft 2025 event. This is a fun draft-style game for selecting and scoring bucking horse colts.

## Project Structure
- `index.html` - Main application file with draft board, scoring, and standings
- `prime-time-christmas-draft-2025.html` - Alternate/backup version of the draft page

## Technical Details
- **Type**: Static HTML/CSS/JS website
- **Server**: Python HTTP server on port 5000
- **No build required**: All code is inline in HTML files
- **No backend/database**: All data is stored in browser localStorage

## Features
- Snake draft system for 8 participants
- Colt information with pedigree data
- Scoring system (1-25 points)
- Leaderboard/standings tracking
- Payout calculations ($96/48/16 for 1st/2nd/3rd)

## Deployment
Static deployment serving the root directory.
